event before a datetime
datetime:{* TO '2021-08-29'}


- get the most recent (or future) date in the db
curl -X POST "localhost:9200/3d459c732a6b49cda9f9eb355b5b364a/_search?size=0&pretty" -H 'Content-Type: application/json' -d'
{
  "aggs": {
    "max_datetime": { "max": { "field": "datetime" } }
  }
}
'

- get the oldest (or future) date in the db
curl -X POST "localhost:9200/3d459c732a6b49cda9f9eb355b5b364a/_search?size=0&pretty" -H 'Content-Type: application/json' -d'
{
  "aggs": {
    "min_datetime": { "min": { "field": "datetime" } }
  }
}
'

- events in a date range
curl -X GET "localhost:9200/_search?pretty" -H 'Content-Type: application/json' -d'
{
  "query": {
    "range": {
      "datetime": {
        "gte": "2021-08-29T00:00:00.000",
        "lte": "2021-08-31T23:59:59.000"
      }
    }
  }
}
'

- count of events in a date range
curl -X GET "localhost:9200/_count?pretty" -H 'Content-Type: application/json' -d'
{
  "query": {
    "range": {
      "datetime": {
        "gte": "2021-08-29T00:00:00.000",
        "lte": "2021-08-31T23:59:59.000"
      }
    }
  }
}
'
- count of events before a date 
curl -X GET "localhost:9200/_count?pretty" -H 'Content-Type: application/json' -d'
{
  "query": {
    "range": {
      "datetime": {
        "lt": "2021-08-31T23:59:59.000"
      }
    }
  }
}
'

- count of events after a date 
curl -X GET "localhost:9200/_count?pretty" -H 'Content-Type: application/json' -d'
{
  "query": {
    "range": {
      "datetime": {
        "gt": "2021-08-31T23:59:59.000"
      }
    }
  }
}
'

Date Ranges 
datetime:[2021-08-29 TO 2021-08-31]
Date prior to 
datetime:[*  TO 2021-08-29]
Dates after
datetime:[2021-08-31 TO *]
Dates either side of a range
datetime:[*  TO 2021-08-29] OR datetime:[2021-08-31 TO *]


data_type:"windows:evtx:record" AND event_identifier:4624 AND xml_string:"/LogonType\"\>3/" AND datetime:[2021-08-29 TO 2021-08-31]
